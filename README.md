# Expense Tracker - Java Project

This is a simple console-based expense tracker written in Java.  
It demonstrates basic Object-Oriented Programming concepts like abstract classes, inheritance, interfaces, and polymorphism.

With this program, you can:

- Add different types of expenses: Food, Travel, and Rent  
- Display all expenses  
- View total expenses and category-wise totals  
- Delete an expense by its ID  
- See total rent
